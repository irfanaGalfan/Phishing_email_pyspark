**OBJECTIVE:** The objective of this analysis is to leverage the Apache Spark distributed computing framework via PySpark to ingest, clean, and analyse a high-dimensional cybersecurity dataset targeted at detecting phishing email indicators.
By applying distributed data engineering techniques and machine learning algorithms (Decision Trees and Logistic Regression), this pipeline handles missing values, profiles key statistical distributions, maps feature interactions and establishes a predictive Phishing Risk Scoring System capable of quantifying threat thresholds.
